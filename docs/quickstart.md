# Quickstart

## 1) Define your operating context (15 min)
- Complete `templates/company-profile.md`.
- Update `company/vision.md`, `company/offers.md`, and `company/kpis.md`.

## 2) Pick one immediate outcome (5 min)
Choose one workflow from `workflows/` based on your highest-priority business goal.

## 3) Activate only the required roles (10 min)
For that workflow, run the Prompt Starter in each participating employee file under `employees/`.

## 4) Enforce founder approvals (ongoing)
Before any external send, legal/financial action, or production release, apply `os/approval-policy.md`.

## 5) Close the loop weekly (30-60 min)
Run `workflows/weekly-founder-review.md`, then update shared memory using `os/shared-memory.md`.
