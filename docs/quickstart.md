# Quickstart

## 1) Initialize your company
- Complete `templates/company-profile.md`.
- Fill files in `company/` with real constraints and goals.

## 2) Pick your first workflow
Choose one from `workflows/` based on your immediate objective.

## 3) Activate employee roles
Use the Prompt Starter in each relevant employee spec inside your agent framework.

## 4) Enforce approvals
Apply `os/approval-policy.md` before external actions.

## 5) Record memory
Update `os/shared-memory.md` conventions after each milestone.

## 6) Run weekly review
Use `workflows/weekly-founder-review.md` every week.
