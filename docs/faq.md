# FAQ

## Do I need a custom runtime?
No. This is markdown-first and can run in any agent environment or manually.

## Is this only for software startups?
No. It also works for consulting, services, and creator-led businesses.

## How strict are approvals?
Strict for external sends, legal/financial actions, and production changes.

## Can I remove roles I do not need?
Yes. Start with the minimum team and expand as needed.

## Where should memory live?
In markdown by default, optionally mirrored into your own databases using provided schemas.
