# Usage Patterns

## Pattern A: Founder-in-the-loop sprint
- Founder sets weekly objective.
- Chief of Staff decomposes tasks.
- Functional employees execute drafts.
- Founder approves releases and sends.

## Pattern B: Parallel pods
- Growth pod: content + SEO + social + ads.
- Product pod: PM + UX + engineering + QA.
- Ops pod: support + success + finance/legal ops.

## Pattern C: Campaign mode
Use when launching product, offer, or major initiative.
- Define campaign brief.
- Assign role owners.
- Run workflow with hard approval gates.
- Publish weekly report.
