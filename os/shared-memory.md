# Shared Memory

## Memory Layers
1. **Company Memory**: vision, offers, policies, voice, ICP.
2. **Customer Memory**: account context, objections, commitments, history.
3. **Execution Memory**: active tasks, decisions, blockers, outcomes.
4. **Learning Memory**: experiments, wins, failures, playbook updates.

## Update Cadence
- After each workflow milestone
- After founder approvals
- At weekly founder review

## Minimum Memory Entry
- Date
- Owner
- Context
- Decision/insight
- Impact
- Next action

## Storage Principle
Keep memory human-readable first (markdown), machine-parseable second (schemas where useful).
