# Example: Solo SaaS (Client Reporting Tool)

## Scenario
A solo founder is building a lightweight reporting product for boutique marketing agencies.

## Starting Point
- Audience: 5-30 person agencies
- Current pain: manual weekly reporting takes 4-6 hours per account
- Founder goal: 3 paying design partners in 30 days

## Workflow Stack
1. `workflows/launch-a-product.md`
2. `workflows/get-first-10-customers.md`
3. `workflows/weekly-founder-review.md`

## 30-Day Execution Plan
- **Week 1:** finalize ICP, offer, and MVP scope; create landing page copy.
- **Week 2:** ship MVP + QA checklist; prepare onboarding and support scripts.
- **Week 3:** run founder-approved outbound to 40 qualified agencies.
- **Week 4:** close first design partners and capture onboarding feedback.

## Expected Artifacts
- PRD + sprint plan
- Landing page and outreach sequence
- Proposal + onboarding checklist
- Weekly KPI report
