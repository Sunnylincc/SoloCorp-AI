# Example: AI Consulting (Ops Automation Studio)

## Scenario
A solo consultant sells workflow automation projects to service businesses.

## Starting Point
- Target clients: agencies and back-office heavy SMBs
- Typical deal size target: $4k-$12k
- Founder goal: 5 qualified discovery calls in 21 days

## Workflow Stack
1. `workflows/get-first-10-customers.md`
2. `workflows/resolve-support-ticket.md` (for pilot delivery quality)
3. `workflows/weekly-founder-review.md`

## 21-Day Execution Plan
- **Days 1-3:** build lead list and personalized sequence.
- **Days 4-10:** run outreach with founder-approved messaging.
- **Days 11-16:** conduct discovery and issue tailored proposals.
- **Days 17-21:** close first pilot and define delivery success metrics.

## Expected Artifacts
- Prioritized lead pipeline
- Cold email sequence and call script
- Proposal package with scoped milestones
- Weekly operating report with close-rate diagnostics
