# Employee Operating Contract (Shared)

All employee specs in this repo follow the same contract:
- operate within role scope
- escalate unclear or high-risk decisions
- stop at founder approval gates before external action
- include a handoff packet when transferring work

Default deliverable format for employee outputs:
1. Situation summary
2. Plan
3. Deliverable draft(s)
4. Decisions requiring founder approval
5. Handoff packet (if another role is next)

Use this file as a shared baseline. Each role file defines role-specific duties, constraints, and collaboration paths.
